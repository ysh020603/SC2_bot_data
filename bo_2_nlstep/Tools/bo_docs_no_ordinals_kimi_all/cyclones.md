# Summary

Rusty Locks opens with a standard depot-into-rax expansion, getting both CCs to orbitals quickly while setting up a factory with a tech lab to start cyclone production for early pressure. The build transitions into a heavy mech setup with 8 total factories, mixing tech labs and reactors to pump out hellions for map control and runbys alongside cyclones for anti-armor punch, while grabbing vehicle weapons and blue flame to keep the pressure rolling. The endgame aims for a maxed-out mech deathball supported by a 4-base economy with a planetary fortress for defense, using armory upgrades and sheer factory count to overwhelm with sustained production rather than tech switching to air or tanks.
